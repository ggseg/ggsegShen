# ggsegShen: Shen 268 Functional Parcellation for the 'ggseg' Ecosystem

Shen whole-brain parcellation of 268 functional nodes for the 'ggseg'
ecosystem. Provides a unified 'ggseg_atlas' object with both 2D polygon
geometry and 3D vertex indices, for use with 'ggseg' and 'ggseg3d'.
